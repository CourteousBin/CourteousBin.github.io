HBuilderX连接模拟器开发与调试

HBuilderX连接模拟器

  1. 下载HBuilderX
  2. 下载夜神安卓模拟器
  3. 第三步:在HBuilderX 中新建一个 uni-app 项目
  4. 打开模拟器
  5. 如果 HBuilder x 未检测到 模拟器，那就需要adb命令
  6. adb命令的使用:找到HBuilder X 的目录 , 找到HBuilder X 带的adb程序，将此程序在的路径复制，放到Path中（环境变量）







7. 运行 cmd
8. 查看 adb 的版本（测试是否能使用）

  1. 使用 adb命令 查看 连接的设备( adb devices )
  1. 使用adb命令 让模拟器连接上电脑 ( adb connect )
