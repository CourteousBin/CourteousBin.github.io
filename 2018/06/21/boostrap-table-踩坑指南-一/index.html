踩坑

  • 用惯 ajax 一般都喜欢看服务器返回的数据，但是，用 bootstrap-table 怎么看呢?
    • responseHandler:function(res)
      • res 就是服务器返回的值，console.log(res) 会出现报错情况，不用理会，查看完数据删除报错自动消失。

  • 我拿到数据怎么渲染去页面?

    • 看到上面 demo 的 columns，field 就是 json 数据里面的 name，设置好头部数据直接渲染到页面。

  • 我要开启条数设置，分页，怎么办？

    • pagination 默认为 false 要手动开启。
