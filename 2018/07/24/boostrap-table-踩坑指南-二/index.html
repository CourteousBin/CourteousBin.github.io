boostrap-table-踩坑指南-二

简介

如果在表格需要用到插件时候，请第一个时间想到 boostrap-table 。

它基本能满足你各种需求，下面我用 boostrap-table 做了一个属性结构图。

在实现的过程中，让我学到很多解决 BUG 的方法。

：）

脸上有微笑，心中MMP。

使用场景


完成一个可以展开的树形结构视图。

可以轻松查看上下级关系，容易管理。

使用步骤

基本知识

  1. 概念:这是 boostrap-table 扩展的内容，需要另外下载插件，boostrap-table 只是做了兼容。

  2. 插件名称：TreeGrid jQuery plugin

  3. GitHub:jquery-treegrid

下载

  1. TreeGrid jQuery plugin

  2. bootstrap-table

基本使用

后台数据结构

踩雷 :)

  1. 后台返回 JSON 数据 pid、id 不能为字符串(String)，一定要数字(Number)。用时（3小时）

  2. jquery-treegrid 依赖的插件版本不能太高，一定要按官方为准。(3小时)

因为 bootstrap-table 并不是实现树形结构的核心内容。

所以 jquery、bootstrap、bootstrap-table 都使用 jquery-treegrid 包自带。

排雷思路

  1. 查看官方文档、DEMO。
  2. 用静态服务器学习使用 DEMO 。
  3. 注意查看前后端传输数据结构。
  4. 注意依赖文件版本问题。
  5. 最好使用 DEMO 提供的依赖文件。

待解决问题

  1. 分页后，树形结构失效。
  2. 依赖插件版本太低,导致小 bug
