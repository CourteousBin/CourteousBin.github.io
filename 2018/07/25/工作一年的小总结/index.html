这一年，我做了什么

发表于 2018-07-25 | 分类于 JavaScript |
不要回头，前面还有很多路

青春流逝得如此之快，再想把握也会从手心悄然而去。

距离毕业已经一年了，回想起去年六月的酷暑，我们在宿舍里描述着各自的未来，谈论着各自理想。总觉得校园束缚了我们，只有踏出社会后，我们能施展才能走向人生巅峰。

但，现实是残酷的。每年六月份毕业生走出校门，如何在茫茫人海里找到哦一份合适的工作成为了我第一个难题。

制作简历，投简历，面试，录取。这个过程我用了足足两个星期，这两个星期无疑对心理的考验。
兴庆的是我终于找到了满意的企业。

“感谢欣赏我的人，我会用行动证明你们眼光”——17年6月26号的动态

一周年过去了，这句话还是深深的刻在我的脑海里。这一年，我一直用行动印证这条动态，对自己、对项目负责。

第一个项目：医疗诊所。俗话说万事开头难，直接被它教做人。在读书时，遇到问题百度都能够解决。唯独它，让我质疑了百度的专业性。
做第一个吃螃蟹的人，做人就要有开拓的精神。百度、必应、谷歌所有搜索引擎用一个遍，面对鱼龙混杂的答案不断尝试，最终，真做成了。螃蟹真香！

第二个项目：小程序。17年初诞生的技术，技术总监交代了需求：“动手做吧~”。除了教会我一脸蒙逼还教会了看技术文档。

第三个项目：商用平台。“你和客户谈需求吧！”和蔼的语气、善良的眼神使我在夏天里感受到冬天的寒意。如此大的项目直接和客户对接，是信任，是责任。在这个过程中，与人交往，项目流程和业务能力无疑是质的飞越。

第四个项目：小程序。“自己找客户谈吧”。有一位志同道合的朋友真的很重要，我们一起找到了第一位客户。他教会了我做方案、思维导图和谈客户。

…

至今，已经完成了七至八个项目。每一个项目都是一位老师，督促我学习，促进成长。

上一年，实习年，这一年，正式开始职业生涯。

该是继续前行的时候了，不要回头，前面还有很多路。
